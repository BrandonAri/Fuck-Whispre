# Third-party notices

## whisper.cpp

This app bundles `whisper-cli` from [ggml-org/whisper.cpp](https://github.com/ggml-org/whisper.cpp), licensed under the MIT License. The complete license is also included inside the built app at `Contents/Resources/whisper.cpp-LICENSE.txt`.

The bundled `ggml-base.en.bin` model is the English Whisper base model converted for `whisper.cpp`, downloaded by the project's official `models/download-ggml-model.sh` script.

Both components run on the user's Mac. The application contains no hosted transcription API integration.
